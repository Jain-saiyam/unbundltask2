Customizable Chocolate Pack
Introduction
This feature allows customers of the Chocolate brand to create their own custom pack of chocolates. Users can select up to 8 different chocolate items, and the total price updates automatically based on their selections.

Features
Custom Selection: Users can build a custom pack of chocolates from a variety of options.
Item Limit: The selection is limited to 8 items to maintain an exclusive and curated experience.
Dynamic Pricing: The total price is calculated and displayed in real-time as users pick their chocolates.
User-Friendly Interface: The design is intuitive and responsive, ensuring a seamless experience on all devices.
Installation
Include the HTML file in your project and ensure the CSS and JavaScript are properly linked. The styles will create a pleasant UI/UX, and the script will manage the selection and pricing logic.

Usage
To add or remove chocolate options, modify the list in the HTML under the #chocolate-list element. Each li element should have data-name and data-price attributes corresponding to the chocolate type and price.

Code Explanation
HTML: Outlines the structure of the chocolate selection interface.
CSS: Provides styling to make the interface aesthetically pleasing and user-friendly.
JavaScript: Contains the logic for selecting chocolates, updating the custom pack, and calculating the total price.
Customization
Customize the look and feel by modifying the CSS file. Add or remove chocolate options by adjusting the HTML.

Compatibility
The feature is designed to be responsive and compatible across different browsers and devices.
